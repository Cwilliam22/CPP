
Divers manipulations de files 

Librairie fstream
#include <fstream>

3 classes inlcude dans la librairie fstream
	ofstream -> Pour créer et lire un fichier
	ifstream -> Pour lire depuis le fichier
	fstream	-> Une combinaison de ofstream et ifstream: créer, lire, et écrire sur un fichier

Exemple :
	// Créer et ouvrir un fichier texte
	std::ofstream MyFile("filename.txt");

	// Écrire dans le fichier
	std::MyFile << "Files can be tricky, but it is fun enough!";

	// Fermer le fichier
	std::MyFile.close();

Comment manipule-t-on un fichier en C++ ? :
	- Utilisation des streams
		std::ifstream : lire un fichier
		std::ofstream : écrire dans un fichier
	- Etape pour lire
		open("nom") -> ouvre le fichier
		std::ifstream file("nom") -> créer et ouvre un fichier
		Vérifier si le fichier est ouvert -> is_open()
		Lire -> getline ou ...
		Fermer le fichier -> close()
	- Fonctions à retenir :
		is_open()
		open(std::string filename)
		getline(in, out)
		close()
		find(std::string s1, size_t pos)
		length()
		insert(size_t pos, std::string s1)
		erase(size_t pos, size_t length)
